# Smart Finance Assistant – Budget Buddy

## Overview

Budget Buddy is a simple finance assistant built using Python, Gradio and Pandas.

The application helps users:

- Upload transaction CSV files
- Analyse spending patterns
- Identify highest spending categories
- Receive financial recommendations
- Calculate savings goals
- Ask basic finance-related questions through a chatbot

## Features

### CSV Spending Analysis
Users can upload a CSV file containing transaction data. The application calculates total spending and identifies major spending categories.

### Savings Calculator
Users can enter their current savings, monthly contribution and target amount to estimate the time required to achieve their savings goal.

### Finance Chatbot
The chatbot provides simple budgeting and savings advice.

## Technologies Used

- Python
- Pandas
- Gradio
- Google Colab
